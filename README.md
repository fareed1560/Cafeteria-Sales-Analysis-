Executive Summary:
  
  This Report represent the comprehensive of the Cafeteria Coffee shop sales data. The findings indicate that beverage items—especially coffee-based products—are the primary revenue drivers. Quantity sold has a strong positive impact on total revenue, while price sensitivity affects certain product categories.
Data Overview:

The dataset contains transactional- level sales records with the following columns:

    •	Date
    
    •	Item
    
    •	Quantity
    
    •	Price Per Unit ($)
    
    •	Total Spent ($)

Each row represents a customer purchase transaction.
